# プロフィール

## 略歴

2012 年からエンジニア・クリエイターとして活動しています。  
設計から運用まで実務経験があり、フロント・バックエンド・アプリまで柔軟に対応可能です。  
インフラ構築・Web/DTP デザイン・2D/3D 素材制作も行えます。
